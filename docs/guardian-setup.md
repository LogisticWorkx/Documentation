# Setting up the play area (Guardian)

Before starting the VR training, you must set up the **play area**.  
This is called the **Guardian**.

The Guardian creates a virtual boundary that prevents the user from walking into walls or objects while wearing the headset.

---

## Minimum space

For the best experience we recommend a space of at least:

**3.5 × 2.5 meters**

Make sure there are **no objects inside the play area**, such as chairs, tables, or cleaning equipment.

---

## 1. Set the floor height

When the headset starts for the first time in a new location, it will ask you to set the floor level.

1. Look at the floor with the headset on.
2. Use the controller to place the virtual floor grid on the real floor.
3. Confirm the floor height.

This ensures the VR environment aligns correctly with the real world.

---

## 2. Draw the Guardian boundary

Next, you create the play area boundary.

1. Hold the controller.
2. Press the trigger button.
3. Draw a line on the floor around the area where the user can move.

Make sure the boundary stays **away from walls and objects**.

---

## 3. Confirm the play area

After drawing the boundary, confirm the setup in the headset.

If the space is too small or incorrectly drawn, you can simply redraw the Guardian.

Once confirmed, the user can safely start the VR training.
