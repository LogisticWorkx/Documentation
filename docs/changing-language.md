# Changing the language

The VR training supports multiple languages.  
You can change the language before starting a training module.

The selected language will be used for the instructions and interface inside the training.

---

## Changing the language in VR

1. Start the VR training.
2. You will begin in the **Lobby**.
3. On the whiteboard, select the **Language** option.
4. Choose the language you want to use.

The training will now continue in the selected language.

---

## Changing the language using the Spectator

Trainers can also change the language from the **Spectator**.

1. Open the Spectator.
2. Connect to the VR headset.
3. Open the **Language** option in the controls.
4. Select the desired language.

The language will update immediately for the trainee in VR.
