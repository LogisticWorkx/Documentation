# Installing the Launcher

The **Launcher** is used to install and update the VR training software and to start the Spectator application.

You can install the Launcher on your own Windows computer if you want to run the training without using a training case.

---

## Requirements

Before installing the Launcher, make sure you have the following:

- A **stable Wi-Fi network**
- A **Windows computer** (preferably a laptop)
- A **USB-C port** to connect the VR headset
- A supported VR headset (currently **Oculus Quest 2**)

The Wi-Fi network must allow both the headset and the computer to connect to each other.

---

## Download the Launcher

1. Go to the **Training Workx portal**  
2. Navigate to **Administration → Downloads**
3. Download the **Launcher Installer**

You will also find your **company access code** on this page.

<img src="/assets/portal-download.png" width="1000">

---

## Install the Launcher

1. Open the downloaded **Launcher Installer**.
2. Follow the instructions shown on screen.
3. The Launcher will install automatically.

Once installation is complete, you can start the Launcher from your computer.

---

## Enter the access code

The first time you open the Launcher, it will ask for an **access code**.

1. Copy the code from the portal download page.
2. Enter the code in the Launcher.
3. Confirm.

The Launcher will now connect to your company account and automatically install the **Spectator software**.

<img src="/assets/portal-code.png" width="1000">

---

## What the Launcher is used for

After installation, the Launcher can be used to:

- Start the **Spectator**
- **Update the VR headset**
- **Update the Spectator software**
- Manage training connections between headset and computer
