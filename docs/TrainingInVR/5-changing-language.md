# Changing the language

The VR training supports multiple languages. You can change the language before starting a training module. The spectator can also change language during the training.

The selected language will be used for the instructions, voice overs, and interface inside the training.

---

## Changing the language in VR

1. Start the VR training.  
2. You will begin in the **Lobby**.  
3. On the whiteboard, select the **Change Language** option.  
4. Choose the language you want to use.  

The training will now continue in the selected language.

![Change language in VR](assets/change-language.png){ width=70% }

---

## Changing the language using the Spectator

Trainers can also change the language from the **Spectator**.

1. Open the Spectator.  
2. Connect to the VR headset.  
3. Open the **Settings > Language** option in the controls.  
4. Choose if you want to change the Spectator or Student language (the Spectator is the language on the box, the student is the language in the headset).  
5. Select the language you want.  

![Spectator language settings](assets/spectator-languages.png){ width=70% }

---

The language will update instantly.
