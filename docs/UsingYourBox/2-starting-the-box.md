# Starting the box

This box allows you to easily stay up to date, see what the trainee is doing in VR, and start training as quickly as possible. The process for starting the box is quite simple.

---

## 1. Plug in the box

Plug the box into a wall socket. This will turn on the box and the screen automatically. Startup will take approximately 2 minutes.

![Box plugged in](assets/box-plugged-in.png){ width=70% }

---

## 2. Optional: Connect a screen

If you want to, you can now connect a screen using the HDMI cable in the box. Simply plug the cable into the right corner of the box. The screen will automatically connect when you start the spectator.

![HDMI input empty](assets/hdmi-empty.jpeg){ width=45% }

![HDMI plugged in](assets/hdmi-pluggedin.jpeg){ width=45% }

---

## 3. Turn on the headset

While you're waiting for the box to start, you can turn on the Virtual Reality headset by pressing and holding the power button on the side. The headset will take a couple of seconds to start up. You can then put on the headset.

![Turn on headset](assets/turn-on-headset.png){ width=70% }

---

## 4. Start the spectator

Now that the box has started up, you can start the spectator by tapping the **Start** button on the screen of the box. Afterwards, you can select the headset that you want to watch.

![Launcher](assets/launcher.png){ width=70% }

---

## 5. Select headset

After the spectator has started, you can now select the headset. Select the headset you want to watch and press **Confirm**.

![Select headset](assets/spectator.png){ width=70% }

---

## 6. Connecting

Your box will now automatically connect to the headset once it comes online. In the meantime, it will display a range of different images while it is loading.

![Connecting screen](assets/spectator-connecting.png){ width=70% }

---

The box is now fully set up and ready to use. All you need to do is put on the headset and start training.

For more information, see [Using Your Box → How to Use VR](3-how-to-use-vr.md).
