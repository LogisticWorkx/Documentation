# Setting up classes

Classes allow multiple students to train using the same VR headset.  
Students within the same class can switch between users during training.

Classes are created and managed in the **Training Workx portal**.

You can access the portal at:

- Logistic Workx: portal.logisticworkx.com
- Cleaning Workx: portal.cleaningworkx.com

---

## Creating a class

1. Log in to the **Training Workx portal**.
2. Go to **Training → Classes**.
3. Click **New Class**.
4. Enter a name for the class.
5. Select the students that belong to this class.
6. Save the class.

Choose a clear name so trainers can easily identify the correct class.

## Assigning a class to a headset

To allow students from a class to train on a headset, the headset must be linked to that class.

1. Go to **Training → Devices**.
2. Select the headset you want to use.
3. Choose the class that will train on that headset.
4. Save the changes.

<img src="/assets/select-class.png" width="500" style="border: 1px solid #e5e5e5; box-shadow: 0 4px 10px rgba(0,0,0,0.1);">

---

## Switching students during training

Once a headset is linked to a class, students can switch users inside the VR training.

This can be done from the **student panel next to the module selection screen**.
