# Using lesson plans

Lesson plans allow trainers to organize which training modules students should complete.  
They make it easier to guide students through a structured training program.

Lesson plans are managed through the **Training Workx portal**.

You can access the portal at:

- Logistic Workx: portal.logisticworkx.com
- Cleaning Workx: portal.cleaningworkx.com

---

## Creating a lesson plan

1. Log in to the **Training Workx portal**.
2. Go to **Training → Lesson Plans**.
3. Click **New Lesson Plan**.
4. Enter a name for the lesson plan.
5. Select the modules you want to include.
6. Save the lesson plan.

## Assigning a lesson plan

Once a lesson plan is created, it can be assigned to students or classes.

1. Open the lesson plan in the portal.
2. Select the students or class that should use it.
3. Save the changes.

Students will now see the assigned modules when starting the VR training.

---

## Using lesson plans in VR

When students start the training, the modules from the lesson plan will appear in the **module selection screen**.

They can then complete the modules in the order defined by the trainer.
